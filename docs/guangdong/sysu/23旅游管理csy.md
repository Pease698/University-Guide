# 23届csy的分享

关于2023级本科生在中山大学那些事

## 1. 关于中山大学南校园

食堂排名：学五和松涛是俩头部，学五总体来说自选比松涛好吃，二楼的菜品也很不错；松涛胜在菜式多而且质量也不差。南草坪食堂稍微有点偏，在一教正门的那条路往前走，不过里面的自选套餐也是相当美味。学一食堂给到拉完了

宿舍楼：各式各样全看抽奖。总的来说：162以及周围皇帝宿舍>181为代表的一系列小北门附近宿舍，剩下的感觉都一般。181总体来说还行，主要是区位太靠近小北门，小北门外面小吃街很不错。而且外卖也好拿。

教学楼通勤：如果要在这里呆四年建议买电瓶车，因为基本上从宿舍区去到逸夫楼教学楼真的很远。不过南校的路太窄容易堵车。不过校内是有充电桩的。一教比较古老，二教蛮新的。都在松涛附近

体育场：一共有西大体育场、英东体育场两个主要体育场。西大体育场旁边就是学一食堂和逸夫楼教学楼，有一个足球场（不知道是不是标准的）；英东体育场就是那种包围式体育场，有高看台，旁边有一个专门的室内排球场（经典室内高手局，想混熟记
得常来）；然后就是皇帝宿舍（162等）和春晖食堂旁边的新体场，左边排球场右边篮球场；新体即新体育馆，在松涛背面正对面，里面可以预约打羽毛球等

## 2. 转专业
中大的转专业政策相当的不错。每个学期下册能转一次，基本如果你努力一点考到本专业前30%，都可以考虑。但是要注意，王牌专业就要难一些，需要你对想转的学院有一定的了解，并且可能还要进行一些拓展学习（比如岭南学院这种）。

本人就读于旅游学院，学习的本来是高数3，难度较低，然后有周围的朋友为了转院自学了完整的高数1。但是总体来说还是相当宽松的。

## 4. 关于中山大学珠海校区
食堂排名：槿园>榕园>若海>荔园。槿园总体都很好，三楼东北人家性价比之王，而且菜味不错，二楼自选也还行。榕园吃得少，不做评价，总体质量还可以。若海现在二楼烧腊还不错，其他的不做评价。荔园只有几个能吃，个人感觉他的自选拉完了，但是一楼的拉面和二楼土耳其烤肉还行，并且拥有kfc。不过星期四要早点出手不然可能会卖光（尤其中午）

教学楼：就一栋不用找。通勤建议自己备自行车，本来学校内拥有庞大数量的共享，不知道是被投诉了还是咋的消失了很多（主要是哈罗），办共享单车月卡建议还是早点起来抢车不然会没车用

体育场：教学楼靠近北门方向有一个体育场，只有标准400跑到和一片小足球场。体育馆里面就一个室内排球场其他的全是羽毛球场。羽毛球场要预约。学校有专门的体育经费的每个月，理论上来说根本花不完，放心使用。

综测与奖学金：本人稍微懒惰，几乎没搞过这个，但是可以说说大概有啥要求。绩点、志愿时、体育章等有要求，班内担任职务或者学生会职务也有加分，以及一些国家级、省级比赛都可以加分，然后评定奖学金。

## 5. 关于中山大学旅游学院and酒店管理系的一些个人见解

中山大学旅游学院分为3个系，规划系、会展系、酒店系。在大一下有志愿填写，然后专业分流。分流是会参考绩点排名的。坦率的说绩点排名为规划>会展>酒店。当然是按大一两学期的绩点综合。大一新生现在都在南校先读一年再分流到各个校区。当然分流了之后绩点排名也就只在各自系里面算。

我只从就读体验来评价的话:

大二课程还比较多，不同系的同学还是除了一些专业必修课之外，其他的课程是可以互选的。然后要学高等数学（大一），学概率论等，大二下就没有数学学了。推荐高俊、刘易、张骁鸣、罗晓燕、钟淑如和兰俊棒老师的课，个人认为比较有趣有思想，而且作业难度不高。杨云老师的课基本不查考勤。建议可以在大一大二多从事一些有意思的比赛，旅院老师指导比赛还是有一手的，比较专业，哪怕是些完全没涉猎过的领域的比赛也建议尝试。

大三旅院基本上期末都没考试，有的找自己选课原因了这没办法。旅院顾名思义，旅游多，大三规划系跟导师组会经常在寒暑假出去调研，西双版纳、黄山等地方都去过；酒店系调研各种酒店、民宿比较多，但其实都是去玩的，享受的过程中带着一些自己的观点去审视思考一下写得出作业就好。作业还是有一定思考深度的要求的。还有一些对全学院开放的一些调研项目可以报名，以及选课的时候可以选一些有外出调研活动的课程（参考课程大纲或者直接问一些学长学姐就好）。

目前想到就这些，也不知道后续还会不会想起来来补充，就这样吧。
