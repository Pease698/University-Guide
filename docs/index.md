# 欢迎来到东华高中毕业生大学指北 🎓

## 📖 文档作用

这是一个由东华高中毕业生共同维护的**大学信息分享平台**。我们收集整理了各地知名高校的真实就读体验、专业介绍、校园生活、报考建议等内容，旨在为学弟学妹们的升学选择提供第一手参考资料。

通过这份指南，你可以收获到包括但不限于：

- **了解真实的大学生活** - 学长学姐的亲身经历和感受
- **获取专业信息** - 各专业的课程设置、学习难度、就业方向
- **规划未来发展** - 深造、就业、出国等多元路径参考

## ⚠️ 免责声明

1. **内容仅供参考**：本站所有内容均来自东华高中毕业生的个人经验分享，代表作者个人观点，不代表学校官方立场。
2. **信息时效性**：大学政策、专业设置、录取标准等会随时间变化，请以各高校官方网站的最新信息为准。
3. **个体差异性**：每个人的体验和感受因个人背景、兴趣、能力而异，请结合自身情况理性参考。
4. **非官方平台**：本站为学生自发维护的非盈利性质文档，与各高校无正式关联。
5. **维护者水平有限**：本站出现的任何 bug 均来自维护者的水平不足，请多多见谅。

## 👀 阅读者使用方法

这一部分为学弟学妹们提供使用本指北书的使用方法。

### 快速导航

通过顶部导航栏选择你感兴趣的地区，查看对应高校的信息，比如：

- **山东** - 山东大学
- **广东** - 中山大学
- **湖北** - 武汉大学
- **甘肃** - 兰州大学

或者在**其他**专栏查看其他有关大学生活的文章。

如果是手机端进入的，点击左上角菜单标签即可找到对应高校信息或者其他专栏。

### 搜索功能

点击页面右上角的 🔍 搜索图标，输入关键词快速查找内容，比如：

- 搜索学校名：如"北京大学"
- 搜索专业名：如"计算机"
- 搜索关键词：如"食堂"、"宿舍"

### 阅读建议

1. **对比多所学校** - 不要只看一所大学，建议横向对比
2. **关注多个维度** - 学术、生活、就业等多方面了解
3. **保持批判思维** - 理性看待内容，结合自身判断
4. **查看官方信息** - 重要决策前务必访问学校官网确认

## ✍️ 编辑者使用方法

我们欢迎所有东华高中的学长学姐参与内容贡献！无论您就读于中国大陆，或者中国港澳台，再或者是国外的大学，无论您的学校层次如何，我们都欢迎您为本站参与内容贡献。

### 内容规范

- **您可提交的内容**：本站希望提供学长学姐们对您所就读的大学的客观条件与主观感受；同时本站还希望还可以提供学长学姐们的一些感悟，这一部分我们会放置到**其他**专栏中，建议以一篇文章的形式展现。（可以参考专栏中已经存在的文章）
- **您贡献的内容务必真实**：如果是与学校客观条件或者主观感受的内容，您攥写的内容请保证**真实性**。同时，应当表明您的年级以保证**时效性**。
- **您贡献的内容请使用结构化组织**：建议使用标题、列表等让内容清晰易读。
- **您贡献的内容不要暴露任何人的隐私**：不要透露个人敏感信息或他人隐私。
- **您贡献的内容不得出现任何违法言论**：请您保持客观理性，避免攻击性言论、辱骂性言论，同时避免涉及任何带有政治的内容。

### 贡献流程

- **编写内容**：请将您需要投稿的内容编辑到 word 文档，或者 markdown 文档中（最好是 markdown 文档）。如果可以，请您在您编写的内容的最开始加上您攥写内容的主题。
- **文档命名**：文档请命名为“东华投稿-您的署名-您在**东华高中**毕业的时间”。您的署名可以是任何您喜欢的内容。
- **发送邮件**：请将文件发送到邮箱 University_Guide@163.com ，主题就是文档名字。

为了保证本站的安全，暂时不支持其他账户直接修改本站的仓库，请多多见谅。

### 审核

- 所有贡献内容需要通过审核才能被上传到文档中
- 维护者会审核内容的**真实性、规范性和准确性**
- 通常会在**周日**统一进行审核上传
