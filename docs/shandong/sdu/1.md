# 山大计算机测如评

本人是2023年参加的高考，最后进入山东大学计算机科学与技术专业学习。这里从我本人的经历讲起，分享一下我们学校的情况。

## 初见

我们学院坐落在山东省青岛市即墨区，为什么我要强调这个区，这是因为我们校区是位于青岛市的郊区地带。在这里乘坐地铁到达市区至少需要坐一个小时的地铁才能到达。

我们学校去到青岛市的一些重要交通枢纽的时间大致为：

1. 青岛北站：1.5h地铁
2. 青岛站：1.5h地铁
3. 胶东国际机场：2-3h地铁

我还没试过打车需要多久，不过要花不少钱。

## 泰山学堂选拔考试

大一印象最深刻的内容就是军训期间进行的泰山学堂选拔考试。泰山学堂算是我们学校的一种特色班级，对于数学、物理、化学、生物学、计算机五个取向招生，我当时报名的是计算机取向。

这个选拔考试分为两个部分：笔试和面试。笔试部分考查你的数理逻辑，当然还有一些算法竞赛中出现的内容。面试部分是最为残酷的，一共面试四轮（2023年）。第一轮和第二轮是学长学姐给我们面试。第一轮面试是三个人为一组，好像提前发了一个全英文的文献让我们提前阅读，然后总结；随后还用英文问了各种数学问题。第二轮面试是跟一位学长面对面交流，也会问你一些计算机相关内容。第三轮和第四轮是教师面试。第三轮是英语口语考试，1v2，你跟两个考官交流对话，交流的内容比较日常，不会太过学术化。第四轮是1vn，你一个人坐在前面，下面不知道多少个老师来拷问你，会问你各种数学、计算机相关的内容。

计算机取向应该是只有二十多个人能被录取，面试环节进了六十多个人，我实力不足并没有成功选拔上。

泰山学堂里面几乎是百分之一百保研（不过我听说大部分都是保研到本校），学习的内容大多数与普通班的差不多，我们的培养方案是不同的，所以学习的东西是不一样的。同时学堂还有末尾淘汰制，好像是在大一下学期期末的时候会淘汰掉成绩排名最后面的同学，然后大二上学期的时候普通班的同学仍然有一次机会可以转入泰山学堂。

我们学校应该还有其他的特色班，具体情况我也不太了解。

## 三千米测试和游泳

这个是山东大学的传统项目。在山东大学需要上两年的体育课，这两年内的四个学期，每一个学期都要进行15次三千米打卡以及最后的三千米考试。

游泳这个项目是我们在大二的时候新加的，据说应该是大一至少要会游50米，大二要会游100米。

## 饮食

坦白说，青岛校区的食堂真的不怎么样，这个事情持续到2025年。去年我们校区的食堂换了一个负责人，食堂有所改善，青岛校区的两个食堂都还有一些不错的菜品。但是作为一个广东人需要向我的学弟学妹们澄清，食堂的粤菜真的很难吃，如果你真的来到我们校区上学，千万不要尝试，想吃粤菜的时候，建议去找你们的老乡去市区聚餐。

学校有一个小型的商业区（真的非常小），里面有一些店铺，一般都支持外卖送到楼下（也仅仅是这些店的食品允许被外卖送到宿舍楼下，其他校外的外卖会送到校门口）。

## 宿舍

青岛校区至少都是四人间，有一些博士生宿舍好像是二人间，不过你们一般住不上。四人间是上床下桌，有独立卫浴，有暖气，南区宿舍有阳台（但是现在被封上了），北区宿舍就没有阳台了。

宿舍有限电，但是你要带台式电脑是可以承担的了的，承担不了的时候跟宿管说一声会帮你扩的。

宿舍的暖气是免费的，不过有些宿舍的暖气貌似不太行，我们宿舍的暖气还是可以的，冬天睡觉的时候甚至会觉得热。空调是二级还是三级能耗，夏天的时候非常耗电。

## 其他

青岛校区比较大，出行可能比较依赖电动车或者是自行车，现在好像禁止新的电动车入内了，学校给旧车上了一些牌子，无牌车不给进学校。校内有公交，但是我没坐过。校内有美团的电动自行车，我有自己的电动车我也没怎么骑过这个。

学校的教学楼还可以，毕竟校区历史也不老，比较无语的是大部分教室的凳子是那种固定好的，你不能调节凳子和桌子的距离，坐久了会比较难受。

学校的图书馆非常大，即使是期末周的时候，也几乎能保证你一个人占两个人的位置（我们这里一般都是这样的，一个人坐两个人的位置，除非有人预约了你的位置）。不过有点不太好的就是，换季的时候空调开的不及时，去年夏天就是期末周快结束了空调才开。但是图书馆的暖气真的很顶。

学校附近就是一个村子，村子里面有很多餐饮店。

学校附近有一个滨海公园。我很坦诚地说，青岛的海真的很好看，你在滨海公园看到的景色跟在栈桥看到的景色我认为区别不大。

青岛的天气不好，时不时会刮大风。这里几乎不下雪（想看雪建议去一百多公里外的威海看）。

## 计算机学院

下面讲一下我们学院的情况，下面的视角仅仅作为一个大三学生的视角。

### 1. 专业分流

准确来说，我被录取到的专业是叫做“计算机类”，在大一上学期结束的时候会进行分流，流向三个专业：计算机科学与技术专业、数据科学与大数据专业、人工智能专业。不过在最新的培养方案中，人工智能专业好像是单独开设了，我也不太了解。我去向是计算机科学与技术专业，也是最多人去的专业。

这三个专业中，我比较了解我们专业，同时对数据那边的情况有所听闻，智能那边我没啥社交，这里就不提了。我们学校所有人上的课分为三种：必修、限选、通选。必修课和限选课都是你们专业培养方案里面的内容，通选课就是那些其他的选修课。限选课就是在大三（或者有些是大一大二就有的），在一些专业课里面，选择你最感兴趣（当然最好过的也可以），修够一定学分就可以了。我们专业跟数据那边的差别就是必修课和限选课的差别，他们的必修课更侧重于数据科学的内容（具体有什么大数据管理等等），而我们更侧重于计算机科学的内容（我们要学数字电路，虽然我也不知道学这个有什么用）。一个很直观的例子就是，他们的机器学习是专业必修课，而我们的是限选课。

对于专业分流，这个我认为去哪个专业其实没有太大的区别。唯一的区别可能是课程的数量，我们专业前期上的课真的很多，相比数据专业来说，但是总的课程数应该也大差不差，计算机类的专业上的课就是非常多。不过这个排课受培养方案的影响非常大，我就记得我大二的时候一天五节课全是专业课，一天十个小时坐在教室简直是折磨。

### 2. 保研率

我们学院的保研率大约是百分之十五往上一点，应该不会超过百分之二十，这个数字是对于三个专业的都是这样，泰山学堂除外，他们百分百保研。

从2022年开始，我们学校的保研成绩需要加上综测成绩，与学业成绩19开。至于这个综测是什么，包括你去参加各种比赛、活动，如果你能比赛拿大奖
发表论文，加分会比较多。

至于保研去向，据我所听说的，还是大部分保研到本校。

### 3. 教学

在我们学院你想挂科，不是一件简单的事情，老师大部分都会海底捞，只要你完成平时布置的任务，期末考试认真对待一下，就没什么问题。

不过我想说的是，学计算机真的要学会自学，光听课是啥都学不到的。多多查博客，多多问AI，多多了解新的知识才是最重要的。

### 4. 好不好？

我这里非常想说的是，对一个学校的某个专业的评价是好或者不好，单单看他那个学科评价是不够的。

你要问我们学校的计算机专业好吗？我会说还行吧，学科评估也是A级，虽然说对比起其他知名985来说可能不太够好，但是站在全国来说也是不错的。学科评估能给你带来什么？一般来说学科评估比较好的学校，那些老师掌握的资源就会更好，如果你跟他做科研做项目，也更容易出成果。（当然，本科生做科研且能做出很好的成果的就是凤毛麟角）

但是我在这个专业学习过，我肯定也能说出我们专业的一些不足，比如培养方案有些课程顺序不合理等等，这些是要参与过学习才能知道的。
