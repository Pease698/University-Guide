# 卡萨布兰卡分享

关于兰州大学，首先它有两个校区，老校区在城关区（市中心），新校区在榆中县夏官营镇，距离老校区40公里，通勤方式只有校车与的士，值得注意的是本科生都会在榆中校区，医学生除外。

此处主要介绍兰州大学榆中校区，先说优点，校园相较于其他大学来说面积不大，而且学校里面有共享单车和共享电动车，意味着去到校园各处都比较方便。整个学校的规划是方方正正的，路很好认。这两年学校在更新宿舍家具，安装空调，所以学生的住宿条件是相对较好的。

学校有五个食堂，分别是嘉禾苑（三层楼，从早上6:30营业到22:00，但是那里预制菜比较多）、芝兰苑（两层，在正餐点营业，优点是比较便宜，而且二楼可以点菜适合聚餐）、玉树苑（三层楼，三楼是清真餐馆，在正餐点营业，正餐时间人相对其他饭堂少一点）、桃李食堂（一层，那里自助餐还行，正餐点营业，缺点是可选择的比较少）、紫芝苑（最新的食堂，23年10月建成，有四层，选择性比较多，离教学楼比较近，但是离学校大部分区域都比较远，可以在四楼像下馆子一样点餐吃），新生刚开学的一个月内可能会觉得很不错，吃多了也是会腻的，现在饭堂预制菜越来越多，在同学里的风评越来越不好了。

作为一所985高校，能提供的科研平台是挺不错的，保研率平均有30%，针对考研来说也挺有优势，学校有考研自主出卷权，考本校面试基本不卡人，考外校至少不需要担心本科学历问题。

针对学校的槽点主要是地理位置问题，学校建在兰州市榆中县，周边较为荒凉，学校里面没有娱乐设施，学校外面只有一些饭馆、小超市、KTV、台球厅、棋牌室等，而且去市区需要乘坐校车1个小时，意味着在物质、娱乐层面的体验要比其他大学要差很多。也是因为地理位置，师资人才流失，学校资金短缺（例如科研经费少、学校设施更迭周期长）的问题长期存在。
